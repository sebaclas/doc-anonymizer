# Guía Rápida — Anonimizador Pro 🚀

¡Hola! Esta es la guía rápida para que domines el programa sin complicaciones. El secreto está en los archivos **Excel**.

### 1. Seleccioná tu archivo (Word o PDF) 📄
Elegí el documento que querés anonimizar. El programa lo va a leer entero (incluyendo cuadros de texto, pies de página y tablas).

### 2. Detectar y Generar Excel 🧠
Al darle a este botón, el programa "busca" nombres, empresas y montos. Te genera un archivo Excel en la misma carpeta que el original. 
**Abrilo y trabajá sobre él:**

- **Columna "Pseudónimo"**: Poné el alias que quieras usar para reemplazar el nombre original. Si lo dejás vacío, el programa no hará nada con esa entidad.
- **Columna "Acción"**: Poné una **'s'** si querés que el cambio se haga. Dejalo vacío si querés que el nombre se quede como está.
- **Columna "Guardar DB"**: ¡Esto es muy útil! Si ponés una **'s'** acá, el programa se "acuerda" de este nombre para siempre. Esta info se guarda en otro Excel al que podés acceder desde el botón "Base de Datos Maestra". La próxima vez que proceses un archivo similar, ya sabrá cómo anonimizarlo y te aparecera ya marcado en el excel particualr. Siempre podes modificarlo ahi.

### 3. Generar Documento 🔄
Una vez que guardaste y cerraste el Excel, volvé al programa y dale al botón verde. ¡Listo! Se creará una copia anonimizada de tu documento.

### 4. Base de Datos Maestra (Tu "Memoria") ⚙️
Como dijimos, en la parte inferior tenés el botón para abrir la base de datos completa. 
- Usalo para limpiar nombres viejos o agregar nombres a mano.
- Usá la columna **"Aliases"** si una persona se escribe de varias formas (ej: "Juan Pérez, J. Pérez").
- Tambien podes agregar varias lineas con el mismo pseudonimo.
- Poné el modo en **"substring"** para siglas que aparezcan pegadas a números o guiones.

---
**Tip Pro:** Si ya tenés el Excel corregido de antes, no hace falta que vuelvas a detectar. Podés usar el botón **"Cargar Excel Existente"** y generar el documento de una.
