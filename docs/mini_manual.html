Guía Rápida — Anonimizador Pro 🚀

¡Hola! Esta es la guía definitiva para que domines el programa sin complicaciones. El secreto está en los archivos de Excel.

1. Seleccioná tu archivo (Word o PDF) 📄

Elegí el documento que querés anonimizar. El programa lo va a leer entero (incluyendo cuadros de texto, pies de página y tablas).

2. Detectar y Generar Excel 🧠

Al darle a este botón, el programa "busca" nombres, empresas y montos. Te genera un archivo Excel en la misma carpeta que el original. Abrilo y trabajá sobre él:

  • Columna "Pseudónimo": Poné el alias que quieras usar para reemplazar el nombre original. Si lo dejás vacío, el programa no hará nada con esa entidad.
  • Columna "Acción": Poné una 's' para que el cambio se haga. Dejalo vacío si querés ignorar ese nombre.
  • Columna "Guardar DB": ¡Esto es clave! Si ponés una 's' acá, el programa "aprende" ese nombre. La próxima vez ya te aparecerá marcado automáticamente.

3. Generar Documento 🔄

Cuando guardes y cierres el Excel, volvé al programa y dale al botón verde. ¡Listo! Se creará tu copia anonimizada en segundos.

4. Base de Datos Maestra (Tu "Memoria") ⚙️

En la parte inferior tenés el botón para abrir la base de datos completa:

  • Usala para limpiar nombres viejos o agregar nombres a mano.
  • Columna "Aliases": Para variantes (ej: "Juan Pérez, J. Pérez").
  • Modo "substring": Para siglas que aparezcan pegadas a números o guion.
Tip Pro: Si ya tenés el Excel corregido de antes, no hace falta volver a "Detectar". Usá el botón "Cargar Excel Existente" y generá el documento directamente.
